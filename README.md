# Edwin Huang — Personal Webpage

A personal single-page website for **Edwin Huang**, featuring a live clock and an elegant editorial design inspired by the [Poveda template](https://builder.hostinger.com/templates?preview=poveda&hideRating=1).

---

## ✨ Features

- **Live Clock** — displays the current time (hours, minutes, seconds, AM/PM) along with the full date and local timezone, updated every second
- **Poveda-inspired Design** — deep olive green palette, serif typography, and a clean 50/50 split-screen layout
- **Smooth Animations** — fade-up entrance animations and a slow-spinning decorative circle
- **Fixed Navigation** — glass-effect top bar with logo and nav links
- **Fully Responsive** — adapts gracefully to mobile screens

---

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | Vanilla CSS (no frameworks) |
| Logic | Vanilla JavaScript |
| Fonts | Google Fonts — Playfair Display & Inter |

## 🚀 Getting Started

No build steps required. Just open the file in any modern browser:

```bash
# Clone the repo
git clone https://github.com/radimon/0303DICDRL.git

# Open in browser
open index.html
```

Or simply double-click `index.html` in your file explorer.

## 🌐 Deployment

This site can be hosted for free via **GitHub Pages**:

1. Go to the repository on GitHub
2. Navigate to **Settings → Pages**
3. Set source to **Branch: main**, folder **/ (root)**
4. Click **Save** — your site will be live at `https://radimon.github.io/0303DICDRL/`
